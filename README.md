# Little London Management System

Phase 1 foundation for the Little London Management System.

## Stack

- Next.js 15
- React
- TypeScript strict mode
- TailwindCSS
- shadcn/ui structure
- Supabase
- Vercel-ready project layout

## Local Setup

1. Install dependencies.
2. Copy `.env.example` to `.env.local`.
3. Fill in the Supabase public URL and anon key.
4. Start the development server.

```bash
npm install
npm run dev
```

## Phase Boundary

This repository currently implements Phase 1 only: foundation and infrastructure. Operational modules such as students, parents, teachers, attendance, payments, invoices, workshops, birthday events, and nursery are intentionally not implemented yet.
